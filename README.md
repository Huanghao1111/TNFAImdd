# Exploring the potential impact of TNF-α inhibition on major depressive disorder risk: Insights from a prospective cohort and genetic evidence—Supplement code

1. Drug-target two-sample Mendelian randomization:
All GWAS summary statistics used in this study have been deposited in Zenodo and are publicly available at: https://doi.org/10.5281/zenodo.18238543.
Detailed information on data sources is provided. The analysis code for the drug-target two-sample MR is provided here.

2. Prospective cohort analysis using the UK Biobank:
Access to UK Biobank data requires formal application and approval in accordance with UK Biobank’s data access procedures.
All analysis must be performed within the UK Biobank Research Analysis Platform. The code provided here is for statistical analysis and figure generation only.
Data preprocessing and variable derivation have been described in detail in the main manuscript.
